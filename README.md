Videoclub Mary (EJ1_ex2_ED, EJERCICIO 1 ENTORNOS)
Este repositorio contiene el software de gestión para el Videoclub Mary, desarrollado por la empresa Lope de Vega SL. Es una aplicación en Java para controlar los alquileres de películas y los datos de los clientes.

-Requisitos
Lenguaje: Java 8 o versión superior.

-IDE: Preferiblemente Eclipse

-Cómo se ejecuta?
Descarga el código o clona el repositorio.
Importa el proyecto en Eclipse mediante File > Import > Existing Projects into Workspace.
Busca la clase principal que contiene el main y dale a Run.

-Estructura del proyecto
El código está organizado en clases para gestionar, las clases son las siguientes:

Clientes: Registro y datos personales.
Películas: Títulos disponibles y categorías.
Alquileres: Lógica para prestar y devolver películas.
